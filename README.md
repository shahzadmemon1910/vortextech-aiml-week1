# vortextech-aiml-week1
Week 1 AI &amp; ML Internship Task - Data Cleaning and EDA using Python

## Data Cleaning and Exploratory Data Analysis (EDA)

### Objective
The objective of this project is to clean a raw dataset and perform Exploratory Data Analysis (EDA) using Python and Pandas.

### Dataset
Titanic Dataset

### Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

### Tasks Performed
- Loaded the Titanic dataset
- Checked for missing values
- Filled missing values using Median and Mode
- Removed the Cabin column due to a high number of missing values
- Checked and removed duplicate rows
- Verified data types
- Generated summary statistics
- Created a Histogram for Age Distribution
- Created a Bar Chart for Gender Distribution

### Files Included
- vortextech-aiml-week1.ipynb
- Titanic_Cleaned.csv

### Conclusion
This project demonstrates the basic steps of data cleaning and exploratory data analysis. The dataset was successfully cleaned, analyzed, and visualized using Python libraries.
